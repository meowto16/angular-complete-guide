# Еще о Renderer

В прошлой лекции мы использовали Angular Renderer чтобы изменить стиль HTML элемента.

Как было объяснено в прошлой лекции, мы должны использовать Renderer для любых манипуляций с DOM.

Конечно можно делать намного больше вещей, чем просто менять стиль элемента через `setStyle()`. 
Подробнее можно глянуть в доке: https://angular.io/api/core/Renderer2
